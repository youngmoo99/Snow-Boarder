# 🏂 Snow-Boarder

<img src="mainmenu.PNG" width="600"/>

Unity 2D 횡스크롤 스노우보드 게임
좌/우 방향키를 통해 캐릭터가 바닥에 머리가 충돌하지 않고 결승점 까지 무사히 골인 해야 하는 게임

---

## 🎮 기능

- 🏔️ S점프 메커니즘이 있는 부드러운 내리막 경사면
- ❄️ 카툰 스타일 겨울 환경
- 🎿 스노우보더 캐릭터 애니메이션
- ⛷️ 물리학 기반의 움직임과 추진력
- 🎵 배경음악 및 사운드 이펙트
